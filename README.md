# MathPilot GitHub Pages

This folder contains a minimal static site for deploying `MathPilot` with GitHub Pages.

## Files

- `index.html`: homepage
- `privacy.html`: privacy policy page
- `terms.html`: terms of service page
- `contact.html`: contact and data deletion page
- `styles.css`: shared styling

## Before publishing

Update these placeholders if needed:
- your custom domain
- any wording that should exactly match your Google Play developer profile
- any legal text that should exactly match your real app behavior

## GitHub Pages quick setup

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings -> Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose the `main` branch and `/ (root)`.
6. Save and wait for the site URL to appear.

## Custom domain

After GitHub Pages is live:

1. In `Settings -> Pages`, set your custom domain.
2. Add the DNS records required by GitHub Pages at your DNS provider.
3. Enable HTTPS in GitHub Pages after DNS resolves.

## Google Play usage

Use your custom domain homepage as the website URL in Google Play Console.
Use your custom domain privacy page as the privacy policy URL.
